# Frontend do CanalGov

Este frontend usa Vue 3 com Vite.

## Ferramentas recomendadas

- VS Code
- Volar
- TypeScript Vue Plugin

## Comandos principais

Instalar dependencias:

```sh
pnpm install
```

Executar em desenvolvimento:

```sh
pnpm dev
```

Gerar build de producao:

```sh
pnpm build
```

Executar testes:

```sh
pnpm test:run
```
