# Roadmap

Este arquivo resume a direcao do produto e os blocos de evolucao priorizados para o CanalGov.

- Consolidar a identidade CanalGov em toda a plataforma.
- Melhorar a cobertura de i18n em portugues do Brasil.
- Ampliar a documentacao da API com OpenAPI/Swagger.
- Fortalecer fluxos de atendimento, automacao, SLA e operacao por equipe.
